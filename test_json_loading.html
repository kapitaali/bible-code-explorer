✅ JSON Validation Checklist

Your JSON files should have:

  • ✓ Valid JSON syntax
  • ✓ "metadata" object with required fields
  • ✓ "books" array with at least one book
  • ✓ Each book has "id", "chapters" array
  • ✓ Each chapter has "verses" array
  • ✓ Each verse has "number" and "text"
