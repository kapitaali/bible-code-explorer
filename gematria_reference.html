Hebrew Gematria

Three systems are provided. Mispar Hechrachi is the classical standard used in rabbinic literature. Mispar Gadol elevates the five final (sofit) letter forms to 500–900. Mispar Katan reduces each letter to a single digit by collapsing place values. Dashed-border cells indicate final (sofit) forms.

Greek Isopsephy

Classical Greek uses a non-consecutive numbering because the original 27-letter alphabet included three archaic letters (Digamma=6, Koppa=90, Sampi=900) that are no longer used. The modern 24-letter alphabet therefore skips 6, 90, and 900. The Bible Code Explorer normalises all text to uppercase and removes diacritics before calculation.

Latin Gematria

Ordinal (A=1) assigns sequential values to all 26 letters and is the most practical system for ELS work. Literae Notae is the historical Roman system where only letters that double as Roman numerals have values (I=1, V=5, X=10, L=50, C=100, D=500, M=1000); all other letters are 0. Grey cells indicate zero-value letters.

English Gematria

Standard / Ordinal (A=1…Z=26) is by far the most commonly used English system. Reduction maps A–I to 1–9, J–R to 1–9, S–Z to 1–8 — essentially a digital root per letter. Reverse inverts the alphabet so Z=1 and A=26.

Finnish Gematria

Finnish uses the same 26 Latin letters plus three extras at the end of the alphabet: Å=27, Ä=28, Ö=29. These are treated as fully distinct letters, not variants. The Reduction system maps all 29 letters to values 1–9 using the same modulo-9 logic applied to the full Finnish alphabet order.
